1. Wymienić oraz opisz dwie części selektora,

Element: Część selektora, która określa element HTML, na którym mają być zastosowane
style. Na przykład, w selektorze p, p jest elementem, który odnosi się do wszystkich
paragrafów w dokumencie.


2. Znaczenie nawisów klamrowym {...},

Nawiasy klamrowe w CSS są używane do grupowania reguł stylów. Wewnątrz nawiasów
klamrowych zawarte są deklaracje stylów, które mają być zastosowane do wybranych
elementów HTML.


3. Opisz cechy,

Specyficznosc, Dziedziczenie, Kaskadowosc, Selektory atrybutów


4. Opisz wartości

Tag HTML, Klasa, Identyfikator, Pseudoklasy, Atrybuty


5. Kiedy stosujemy znak średnika (;), kiedy nie jest konieczny

Znak średnika (;) w CSS jest używany do oddzielenia poszczególnych deklaracji stylów od siebie. Jest to wymagane po każdej deklaracji, z wyjątkiem ostatniej w bloku stylów.
